# Platinum One Logistics Starter

This is a clean Next.js starter scaffold for the Platinum One Logistics rebrand.

## Included
- Next.js App Router + TypeScript
- Tailwind CSS
- Framer Motion dependency
- Brand token structure in `/brand`
- Multi-page routes for Home, About, Services, Industries, Rates, Gallery, and Contact
- Placeholder logo copied to `public/logo/platinum-one-logo.png`

## Run locally
```bash
npm install
npm run dev
```

## Suggested next steps
1. Replace placeholder copy with final brand messaging.
2. Add real images under `public/images`.
3. Add the rate sheet under `public/docs`.
4. Wire the contact page to your preferred form backend.
